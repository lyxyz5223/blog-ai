# 📝 博客文章管理指南

## 文章管理系统概述

你现在有两种方式管理博客文章：
1. **直接编辑 JavaScript 文件** - 简单但需要重启服务器
2. **使用管理后台** - 实时编辑，更方便

---

## 方式一：直接编辑文件（推荐初学者）

### 文件位置
```
src/data/blogsData.js
```

### 文章数据结构

```javascript
{
  id: 唯一标识,                    // 数字，必须唯一且递增
  title: '文章标题',                // 20-60 字为佳
  excerpt: '摘要...',              // 50-100 字，显示在列表页
  content: `# 文章内容`,           // 支持 Markdown 格式
  date: '2024-04-01',             // YYYY-MM-DD 格式
  category: '分类',                // React, JavaScript, CSS 等
  author: '作者'                    // 你的名字
}
```

### 快速操作

#### ✅ 添加新文章
```javascript
export const blogsData = [
  // ... 现有文章 ...
  
  {
    id: 5,
    title: '我的新文章',
    excerpt: '这是一篇新文章的摘要',
    content: `
# 我的新文章

## 第一部分
内容...

## 第二部分
内容...
    `,
    date: '2024-04-01',
    category: 'React',
    author: 'Your Name'
  }
]
```

#### ✏️ 编辑现有文章
直接修改对应文章对象中的属性。修改后保存文件，开发服务器会自动热更新。

#### 🗑️ 删除文章
从数组中删除整个文章对象。

### 文章内容 Markdown 语法

```markdown
# 一级标题
## 二级标题
### 三级标题

**粗体文本** 或 __粗体文本__
*斜体文本* 或 _斜体文本_

- 列表项 1
- 列表项 2
- 列表项 3

1. 有序列表 1
2. 有序列表 2
3. 有序列表 3

[链接文本](https://example.com)

> 这是一段引用

\`\`\`javascript
// 代码块
const hello = () => {
  console.log('Hello World');
}
\`\`\`

\`\`\`jsx
// React 代码
import React, { useState } from 'react'

function Counter() {
  const [count, setCount] = useState(0)
  return <button onClick={() => setCount(count + 1)}>{count}</button>
}
\`\`\`
```

## 方式二：使用管理后台（更推荐）

### 访问管理后台

1. **启动开发服务器**
   ```bash
   cd e:\1JsProjects\blog
   npm run dev
   ```

2. **打开浏览器**，点击导航栏中的 **📝 管理** 按钮，或直接访问管理页面

### 管理后台功能

#### 左侧列表区域
- 显示所有文章列表
- 点击文章进行编辑
- 显示发布日期和分类

#### 中右侧编辑区域

**创建新文章**
1. 点击 "➕ 新建文章" 按钮
2. 填写表单：
   - **标题** - 必填
   - **发布日期** - 默认为今天
   - **分类** - 必填（自动创建新分类）
   - **作者** - 必填
   - **摘要** - 必填（显示在列表页）
   - **内容** - 必填（支持 Markdown）
3. 点击 "💾 保存"
4. 根据提示，同步更新 src/data/blogsData.js

**编辑文章**
1. 在左侧列表点击要编辑的文章
2. 右侧显示编辑表单
3. 修改内容
4. 点击 "💾 保存"
5. 同步更新 src/data/blogsData.js

**删除文章**
1. 编辑文章后
2. 点击 "🗑️ 删除" 按钮
3. 确认删除
4. 同步更新 src/data/blogsData.js

#### 导出数据
1. 点击 "💾 导出数据" 按钮
2. 会下载更新后的 `blogsData.js` 文件
3. 替换 `src/data/blogsData.js` 内容

- CSS

## 📱 最佳实践

### 标题
- ✅ 清晰明确，15-30 字最佳
- ❌ 避免过长或含糊不清

### 摘要
- ✅ 简明扼要，50-100 字
- ✅ 吸引人的开头，激发阅读欲望
- ❌ 不要复制文章内容

### 内容
- ✅ 结构清晰，合理使用标题分层
- ✅ 代码块做好语言标注
- ✅ 配合图片和链接（如需要）
- ✅ 定期更新高价值文章

### 日期
- ✅ 使用发布日期，便于档案管理
- ✅ 保持按时间倒序

### SEO 建议
- 标题中包含关键词
- 摘要中明确表达文章核心
- 内容层级清晰（h2, h3 等）
- 使用有意义的链接文本

## 常见问题

### Q: 修改了文章后没有反应？
A: 检查是否正确保存了 `src/data/blogsData.js` 文件。开发服务器应该会自动热更新。

### Q: 如何快速找到某篇文章？
A: 在博客列表页使用搜索框，或在管理后台的左侧列表查找。

### Q: 分类可以自定义吗？
A: 可以。只需在文章的 `category` 字段中填入新分类名，系统会自动识别。

### Q: 如何删除分类？
A: 删除所有使用该分类的文章，分类会自动消失。

### Q: 支持上传图片吗？
A: 当前版本不支持。建议使用外部图床（如 阿里云 OSS、七牛云）的 URL。

### Q: 如何在文章中嵌入视频？
A: 在 content 中使用 HTML：
```html
<iframe width="560" height="315" src="https://www.youtube.com/embed/xxx"></iframe>
```
